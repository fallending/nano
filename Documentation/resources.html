Nano Resources

Some additional resources for use with Nano are provided within the Support folder.

If you would like to contribute additional resources that could be used by applications built with Nano, please see Contributing Code.

Images

This directory contains images that can be used, without credit, by any application that uses Nano.

This includes standard toolbar icons, with a similar appearance to Apple's standard (but proprietary) icons.

Scripts

This directory contains useful scripts related to Nano.

This includes scripts to build the Nano library and demo applications, or to install the Xcode templates and gdb macros.

Xcode

This directory contains resources related to Xcode.

These include the standard Nano .xcconfig files, and templates that can be installed within Xcode to simplify the creation of a new Nano-based application.
