Nano Resources

Some additional resources for use with Nano are provided within the Support folder.

If you would like to contribute additional resources that could be used by applications built with Nano, please see Contributing Code.

Qt

Contains xcodeqt, a tool for Xcode that generates intermediate source for Qt as a build phase. This allows you to use a native Xcode project for Qt development, avoiding the need to use qmake.

For instructions on use, please see the documentation within the script.

Xcode

Contains resources related to Xcode.

These include the standard Nano .xcconfig files, and custom data views that can be added to Xcode to allow Nano types to be viewed within the debugger.
