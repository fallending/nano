Welcome to Nano!

Nano is a C++ library for developing high-performance applications for Mac OS X, Windows, or the iPhone.

Nano provides a consistent set of fundamental data types and core objects for cross-platform application development, as well as classes to integrate with platform-specific APIs.

Nano is Open Source, and distributed under the BSD licence.

The current release of Nano is available here. To get started with Nano, please see the Building Nano guide or read about the available features.
