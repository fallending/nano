Welcome to Nano!

Nano is a C++ library for developing high-performance applications for Mac OS X, iOS, and Microsoft Windows.

Nano provides a cross-platform set of fundamental data types such as strings and dictionaries, and unified interfaces to platform-specific APIs like application preferences.
