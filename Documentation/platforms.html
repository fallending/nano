Supported Platforms

Nano supports the following platforms:

  • Mac OS X (10.5 or later)
  • iPhone OS (2.0 or later)
  • Microsoft Windows (XP, Vista, or Windows 7)

All features are available on all platforms, and the internal platform-specific implementation is not exposed as a public interface.
